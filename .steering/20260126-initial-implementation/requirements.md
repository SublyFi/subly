# 要求内容

## 概要

Business Dashboard（事業者ダッシュボード）の初回実装。事業者がウォレット接続でログインし、サブスクリプションプランの作成・管理、契約数の確認ができるWebアプリケーションを構築する。

## 背景

SublyはSolana上のプライバシーファーストなサブスクリプション決済プロトコル。事業者がプランを作成・管理し、ユーザーがプライバシーを保護した状態でサブスクリプション契約を結べる仕組みを提供する。

Business Dashboardは事業者側のインターフェースとして、以下を実現する：
- プランの作成・編集・無効化
- 契約数の確認（個人は特定できない）
- SDK統合情報の提供

**注意**: 初回実装ではUIとモックデータで動作確認を行い、実際のオンチェーン連携は後続フェーズで実装する。

## 実装対象の機能

### 1. ウォレット接続機能
- Phantomウォレットでの接続
- 接続状態の表示
- Devnetネットワークへの接続

### 2. 事業者登録画面
- 未登録事業者の登録フォーム
- 事業者名の設定

### 3. ダッシュボードホーム
- プラン一覧へのナビゲーション
- 統計情報へのナビゲーション
- 設定へのナビゲーション

### 4. プラン管理機能
- プラン一覧表示
- 新規プラン作成（名前、説明、価格、課金周期）
- プラン編集
- プラン無効化
- 契約数の表示（個人は特定できない）

### 5. 設定画面
- SDK統合情報の表示（Program ID等）
- 事業者プロフィール表示

## 受け入れ条件

### ウォレット接続
- [ ] Phantomウォレットで接続できる
- [ ] 接続中はウォレットアドレスが表示される
- [ ] 切断ボタンで切断できる

### 事業者登録
- [ ] 未登録の場合、登録フォームが表示される
- [ ] 事業者名を入力して登録できる
- [ ] 登録済みの場合はダッシュボードに遷移する

### プラン管理
- [ ] プラン一覧が表示される
- [ ] 「新規プラン作成」ボタンでプラン作成画面に遷移できる
- [ ] プラン名、説明、価格（USDC）、課金周期を入力できる
- [ ] 課金周期は時間単位、日次、週次、月次、年次から選択できる
- [ ] 作成したプランが一覧に表示される
- [ ] プランの編集ができる
- [ ] プランの無効化ができる
- [ ] 各プランの契約数が表示される

### 設定
- [ ] SDK統合情報（Program ID）が表示される
- [ ] 事業者プロフィールが表示される

## 成功指標

- 事業者がウォレット接続から5分以内にプラン作成できる
- UIが直感的で、ドキュメントなしで基本操作ができる

## スコープ外

以下はこのフェーズでは実装しません：

- 実際のオンチェーントランザクション（モックで代替）
- membership-sdk統合（モック/スタブで代替）
- Arcium MPC連携
- Light Protocol連携
- MagicBlock PER連携
- 制裁チェック機能
- API Key生成機能
- 本番環境デプロイ
